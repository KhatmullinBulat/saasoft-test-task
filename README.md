# Тестовое задание (Vue 3 + Vuetify)

## Возможности
- Добавление и удаление учетных записей
- Валидация полей формы (Vuetify rules)
- Хранение данных в `localStorage` и восстановление при перезагрузке
- FSD

## Стек
- Vue 3
- Vuetify 3
- Pinia
- Vite
- TypeScript

## Запуск
```bash
npm install
npm run dev
```

Приложение доступно по адресу `http://test.khatmullinbr.ru`.

## Скрипты
- `npm run dev` — dev сервер
- `npm run build` — production build
- `npm run preview` — предпросмотр сборки
- `npm run lint` — lint
- `npm run lint:fix` — lint с автофиксом
